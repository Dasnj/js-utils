# Jstool
前端常用代码集合

## 安装使用
直接下载 `bulid` 目录下的 [jstool.min.js](https://github.com/Dasnj/jstool/blob/master/build/jstool.min.js) 使用，支持UMD通用模块规范

``` html
  <script src="../jstool.min.js"></script>
  <script>
      jstool.api()
  </script>
```

    - [throttle](./src/function/throttle.js)&emsp;&emsp;节流
